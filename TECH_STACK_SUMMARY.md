### 4. Context Relay机制

**简介**：解决Agent在Session重启、Sub-agent边界、Cron/Heartbeat隔离时的记忆断裂问题。

**核心原理**：
- 文件是唯一的真相源
- 每个执行单元启动时从文件读取context
- 不依赖session记忆
- PROJECT.md + state.json + decisions.md

**应用场景**：
- 跨会话任务连续性
- Sub-agent协同工作
- 定时任务状态保持

**成熟度**：⭐⭐⭐⭐（熟练掌握，核心机制）
